## 呼び出し方
<repositories>
  <repository>
      <id>jitpack.io</id>
      <url>https://jitpack.io</url>
  </repository>
</repositories>

<dependencies>
  <dependency>
    <groupId>com.github.gn5r</groupId>
    <artifactId>Speech2Text</artifactId>
    <version>-SNAPSHOT</version>
  </dependency>
</dependencies>

上記をpom.xmlに追記することで、ライブラリとして使うことができます。
